Final Year Project - CSE'16
Automatic Structuring of Textual System Logs

This tool can be used to generate a structured log report when given a raw log file as an input.

A log line in a log file can have :

• headers - describing pre-configured info in a structured way

• natural language log message - event in human readable form

For Example :

Raw log line :


2019-11-18 20:18:29,467 [INFO] distributedConsensus.LeaderCandidate  Participated to a new round of number 13 with rank as 263

Header fields 2019-11-18 20:18:29,467 [INFO] distributedConsensus.LeaderCandidate
Natural language log message Participated to a new round of number 13 with rank as 263

Identifying the system event for each log message and generating a representative templates for identified events is the problem which this tool tries to address

To do that the tool requires :


• Configuration file with header structure

• Raw log file
