Research and Development Project
vue4logs - Automatic Structuring of Textual System Logs

This tool can be used to generate a structured log report when given a raw log file as an input.

A log line in a log file may consist of,

• Header fields - Describing pre-configured set of fields in a structured manner

• Natural language log message - Describing system event in natural language

For Example,

Raw log line :


2019-11-18 20:18:29,467 [INFO] distributedConsensus.LeaderCandidate  Participated to a new round of number 13 with rank as 263

Header fields 2019-11-18 20:18:29,467 [INFO] distributedConsensus.LeaderCandidate
Natural language log message Participated to a new round of number 13 with rank as 263

This tool is able to identify the system events of log messages and generate a representative templates for identified events.

To do that the tool requires,


• A raw log file

• A configuration file with header structure

Sample Configuration File : ( config.txt )


{ 'log_file' : 'Spark' , 'log_format' : '[<Time>] [<Level>] <Content>' }
